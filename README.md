# Babar Azam Tribute Website

A responsive single-page website dedicated to Pakistan's cricket captain Babar Azam.

## Features
- Complete career statistics
- Timeline of achievements
- Brand endorsements
- Photo gallery
- Fully responsive design
- Smooth animations

## Deployment

This website is automatically deployed to GitHub Pages using CI/CD.

## Live Demo
Visit: [https://zainbinimran.github.io/babar-azam-blog](https://https://zainbinimran.github.io/Blog/)

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- GitHub Pages
- GitHub Actions CI/CD
